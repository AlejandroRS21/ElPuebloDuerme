# Card Assets

This directory should contain card images for all game roles.

## Required card images:
- mafia.png - Mafia role card
- doctor.png - Doctor role card  
- detective.png - Detective role card
- villager.png - Villager role card
- card-back.png - Back of card (for hidden roles)

## Specifications:
- Size: 300x450px (2:3 ratio)
- Format: PNG with transparency
- Style: Should match the dark/horror theme
- Include role name and description on card
